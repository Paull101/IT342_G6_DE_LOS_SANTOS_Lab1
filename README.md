## ğŸš€ Setup and Installation

### 1. Database Configuration

1. Open **XAMPP** and start **Apache** and **MySQL**.
2. Go to `localhost/phpmyadmin` and create a database named `auth_system_db`.
3. The tables will be automatically generated by Hibernate when the backend starts.

### 2. Backend Setup (Spring Boot)

1. Navigate to the `/backend` folder.
2. Ensure `src/main/resources/application.properties` is configured with your MySQL credentials.
3. Run the application using your IDE or via command line:

```bash
./mvnw spring-boot:run

```

### 3. Frontend Setup (ReactJS)

1. Navigate to the `/web` folder.
2. Install dependencies:

```bash
npm install

```

3. Start the development server:

```bash
npm start

```
